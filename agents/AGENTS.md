# AI Agent Instructions

This repository contains rules, best practices, and memory contexts for AI agents working in this project.
It will include states, memories, and details needed to bootstrap an agent to work on the Forge codebase.

For now, refer to the milestone documents in `docs/` and adhere to architectural decisions documented in `docs/DECISIONS.md`.

When testing locally, be sure to use `make e2e` as the single consolidated test command.
